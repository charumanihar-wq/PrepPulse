PrepPulse is a lightweight, AI-assisted self-assessment tool that helps students
measure their interview readiness in under 2 minutes.

## Problem
Students often realize gaps in interview preparation only after failed interviews.
PrepPulse helps identify strengths, gaps, and next steps **before** facing recruiters.

## Key Features
- Role-specific interview readiness scoring
- Overall readiness score (0â€“100)
- Beginner / Intermediate / Interview Ready levels
- Estimated time to become interview-ready
- Personalized improvement suggestions
- Clean, fast, and student-friendly UI

## How It Works
1. User selects target role
2. Self-rates technical skills, resume strength, communication, and portfolio
3. PrepPulse applies role-aware scoring logic
4. User receives score, level, gaps, and next steps instantly
